Introducción

Bienvenido al sistema SmartPOS. Este manual le guiará a través de las funciones principales.

Cómo realizar una venta
  1. Diríjase al módulo de Ventas.
  2. Seleccione los productos o escanee el código de barras.
  3. Seleccione el cliente (opcional).
  4. Haga clic en "Cobrar" y seleccione el método de pago.
